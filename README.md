# tiposdadosav-calculadora

## Regras de Negócio:

Você deve deselver uma classe para calcular os descontos nos valores dos produtos de uma loja durante uma promoção de "queima de estoque". Os descontos devem seguir as regras abaixo:

- Deve retornar o valor com 30% de desconto se o valor for maior que R$1000,00
- Deve retornar o valor com 25% de desconto se o valor for maior que R$500,00
- Para outros valores, deve retornar o valor com 10% de desconto
- Se o valor é negativo, deve apresentar uma mensagem de erro: "O valor deve ser positivo"

*Formulário de Avaliação:* https://forms.gle/3rJXUb1XA76FJQYT6
